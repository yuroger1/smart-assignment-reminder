# Smart Assignment Reminder

## Project Description
Smart Assignment Reminder is a dynamic web application that helps university students manage assignments, deadlines, and reminders.

## Target Users
University students who take multiple courses and need a centralized system to track their coursework.

## Main Features
- User registration and login
- Assignment dashboard
- Add, edit, delete assignments
- Mark assignments as completed
- Search and filter assignments
- In-app deadline reminders
- MySQL database integration

## Technology Stack
- Frontend: HTML5, CSS3, JavaScript, Bootstrap
- Backend: Node.js, Express.js
- Database: MySQL
